# UniqueVibesMX - Página Web Oficial

Página web oficial de UniqueVibesMX - Diseño, Personalización y Recursos Digitales.

## 🎨 Características

✅ **Diseño Responsivo** - Compatible con dispositivos móviles, tablets y desktop
✅ **Modales Interactivos** - Términos y Condiciones, Indicaciones y Descarga
✅ **Diseño Moderno** - Gradientes púrpuras, animaciones suaves y efectos visuales
✅ **Fácil de Personalizar** - HTML, CSS y JavaScript organizados
✅ **Rendimiento Optimizado** - Carga rápida y sin dependencias externas

## 📁 Estructura del Proyecto

```
uniquevibes-web/
├── index.html       # Estructura HTML principal
├── styles.css       # Estilos CSS
├── script.js        # Funcionalidades JavaScript
└── README.md        # Este archivo
```

## 🚀 Cómo Usar

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/perlasoto981/uniquevibes-web.git
   cd uniquevibes-web
   ```

2. **Abrir en el navegador:**
   - Simplemente abre `index.html` en tu navegador favorito
   - O usa un servidor local (Live Server en VS Code)

## 🎨 Personalización

### Cambiar Colores
Edita los colores púrpura en `styles.css`:
```css
--primary-color: #7a2cff;  /* Color púrpura principal */
--dark-bg: #090909;        /* Fondo oscuro */
```

### Cambiar Textos
Edita el contenido en `index.html` directamente.

### Agregar Archivo de Descarga
Cambia la ruta del PDF en `index.html`:
```html
<a href="tu-archivo.pdf" download class="btn-download-modal">
```

## 📱 Responsive
